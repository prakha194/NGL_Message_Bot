# NGL Telegram Bot

A Powerful Telegram bot that sends anonymous messages to NGL links.

## Features
- Send messages to NGL links
- Admin can Schedule message
- AI-generated messages via Gemini API
- Custom Mesaages
- Rate limiting to Non- Premium Users
- Admin can send Unlimited messages 
- Message tracking In real-time 
- Even NGL Premium User cant guess Your Real location,Device info,Etc...

## Environment Variables
- `BOT_TOKEN`: Telegram bot token
- `GEMINI_API_KEY`: Google Gemini API key   
- `ADMIN_ID`: Your Telegram user ID
